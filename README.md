# CodeAlpha Artificial Intelligence Projects

Welcome to my repository of Artificial Intelligence projects developed during my internship at **CodeAlpha**. This repository showcases practical implementations of Natural Language Processing (NLP), Conversational AI, and Generative AI concepts through three real-world projects assigned as part of the CodeAlpha AI Internship Program.

---

## 📌 Projects Included

### 🌐 Task 1: Language Translation Tool

A multilingual translation application that enables users to translate text between different languages using an external translation API.

#### Features
- User-friendly translation interface
- Source and target language selection
- Real-time language translation
- Accurate translation output display
- Supports multiple languages

#### Technologies Used
- Python
- Google Translate API / Translator Library
- Tkinter (GUI)

#### Files
- `Language_Translation_app.py`
- `Language_Translation_Tool_Output.png`

---

### 🤖 Task 2: Chatbot for FAQs

An FAQ chatbot that responds to user questions by matching them with the most relevant pre-defined question-answer pairs using Natural Language Processing techniques.

#### Features
- FAQ-based conversational assistant
- User query preprocessing
- Similarity-based answer retrieval
- Interactive chatbot experience
- Fast response generation

#### Technologies Used
- Python
- NLTK
- Scikit-learn
- Cosine Similarity

#### Files
- `FAQ_chatbot.py`
- `FAQ_Chatbot_output.png`

---

### 🎵 Task 3: Music Generation using AI

An AI-powered music generation system capable of learning musical patterns from MIDI files and generating new musical compositions.

#### Features
- MIDI file processing
- Musical note sequence extraction
- AI-generated music creation
- MIDI output generation
- Automated composition workflow

#### Technologies Used
- Python
- Music21
- TensorFlow / Keras
- NumPy

#### Files
- `music.py`
- `generated_output.mid`
- `music_pyfile_output.txt`
- `midi_songs/`

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/dhyanisatvik884-rgb/CodeAlpha_Artificial_Intelligence_Projects.git
cd CodeAlpha_Artificial_Intelligence_Projects
```

### 2. Install Dependencies

```bash
pip install numpy nltk scikit-learn tensorflow keras music21 googletrans
```

### 3. Run Any Project

#### Language Translation Tool

```bash
cd Task_1_Language_Translation_Tool
python Language_Translation_app.py
```

#### FAQ Chatbot

```bash
cd Task_2_Chatbot_for_FAQ
python FAQ_chatbot.py
```

#### Music Generation using AI

```bash
cd Task_3_Music_Generation_using_AI
python music.py
```
